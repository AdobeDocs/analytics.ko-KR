# 미디어 재생 시간 패널

Media Analytics 고객은 재생 시간을 분석하여 최대 동시 시청 시간 발생 위치 또는 드롭오프가 발생한 위치를 파악하여 컨텐츠 및 뷰어 참여의 품질에 대한 중요한 통찰력을 제공하고 볼륨 또는 규모에 대한 문제 해결 또는 계획에 도움을 줄 수 있습니다.

Analysis Workspace에서 재생 시간 은 특정 시점에 미디어 스트림을 보는 데 걸린 시간이며, 일시 중지, 버퍼 및 시작 시간을 포함합니다.

미디어 재생 시간 패널을 사용하면 최대 동시 시청 시간 및 분류 및 비교 기능에 대한 세부 사항을 사용하여 시간에 따른 재생을 분석할 수 있습니다. 미디어 재생 체류 시간 패널에 액세스하려면 Media Analytics 구성 요소가 활성화된 보고서 세트로 이동합니다. 그런 다음 맨 왼쪽에 있는 패널 아이콘을 클릭하고 패널을 Analysis Workspace 프로젝트로 끌어옵니다.

이 패널에는 24시간 이내에 선택하고 표시할 수 있는 달력의 새 기능도 포함되어 있습니다. 전체 패널에 대해 그렇게 하거나, 연속된 기간을 사용하여 세그먼트를 만들어 프로그램 또는 프로그램의 섹션에서 시청자 리드 인/리드 아웃을 추적할 수 있습니다. 해당 날짜 세그먼트 중 적어도 두 개를 배치하면 공통 x축 시작을 사용하여 라인을 오버레이하거나 특정 x축 시작과 함께 순서대로 표시하는 날짜 시퀀스 표시에 대한 라디오 단추 옵션이 표시됩니다.

## 패널 입력 {#Input}

다음 입력 설정을 사용하여 미디어 재생 시간 패널을 구성할 수 있습니다.

| 설정 | 설명 |
|---|---|
| 패널 날짜 범위 | 패널 날짜 범위 기본값은 오늘입니다. 단 하루 또는 여러 달이 보이도록 편집할 수 있습니다.<br>이 시각화는 1440개의 데이터 행으로 제한됩니다(예: 분 단위 세부 기간에서 24시간). 날짜 범위와 세부 기간 조합의 결과 행이 1440개를 초과하는 경우 전체 날짜 범위를 수용하도록 세부 기간이 자동으로 업데이트됩니다. |
| 세부기간 | 세부 기간 기본값은 분입니다.<br>이 시각화는 1440개의 데이터 행으로 제한됩니다(예: 분 단위 세부 기간에서 24시간). 날짜 범위와 세부 기간 조합의 결과 행이 1440개를 초과하는 경우 전체 날짜 범위를 수용하도록 세부 기간이 자동으로 업데이트됩니다. |
| 패널 요약 번호 | 재생 체류 시간에 대한 날짜 또는 시간 세부 사항을 보려면 요약 번호를 사용할 수 있습니다. 최대값은 최대 동시 시청에 대한 세부 정보를 표시합니다. 최소값은 저점에 대한 세부 정보를 보여 줍니다. 합은 선택 항목에 대한 총 재생 시간을 추가합니다. 패널 기본값은 최대값만 표시되지만 최소, 합계 또는 세 개의 조합을 표시하도록 변경할 수 있습니다.<br>분류를 사용하는 경우 각각에 대한 요약 번호가 표시됩니다. |
| 시리즈 분류 | 필요에 따라 세그먼트, 차원, 차원 항목 또는 날짜 범위별로 시각화를 분류할 수 있습니다.<p>- 한 번에 최대 10개의 줄을 볼 수 있습니다. 분류는 단일 수준으로 제한됩니다.</p><p>- 차원을 끌어오면 선택한 패널 날짜 범위를 기반으로 최상위 차원 항목이 자동으로 선택됩니다.</p>- 날짜 범위를 비교하려면 2개 이상의 날짜 범위를 시리즈 분류 필터로 끌어옵니다. |
| 시간 형식 | Hours:Minutes:Seconds(기본값) 또는 Minutes(정수로 표시되며 .5로 반올림됨)에서 보낸 재생 시간을 볼 수 있습니다. |
| 날짜 시퀀스 표시 | 시리즈 분류로 두 개 이상의 날짜 범위 세그먼트를 배치한 경우 오버레이(기본값) 또는 순차적 중 하나를 선택하는 옵션이 표시됩니다. 오버레이에는 공통 x축 시작을 갖는 선이 표시되므로 선이 병렬로 실행되고 순차적 페이지는 해당 특정 x축 시작을 갖는 라인을 표시합니다. 데이터가 줄 위로(예를 들어, 세그먼트 1은 오후 8:44에 끝나고 세그먼트 2는 오후 8:45에 시작)되면 선은 순서대로 표시됩니다. |

## 패널 출력 {#Output}

[미디어 재생 시간] 패널은 최대, 최소 및/또는 재생 시간 합에 대한 세부 사항을 포함하도록 라인 차트와 요약 번호를 반환합니다. 패널 맨 위에는 선택한 패널 설정을 알려 주는 요약 줄이 제공됩니다.

언제든지 오른쪽 상단의 편집 연필을 클릭하여 패널을 수정하고 다시 만들 수 있습니다.

시리즈 분류를 선택한 경우 꺾은선형 차트에 선과 요약 번호가 각각 표시됩니다.
