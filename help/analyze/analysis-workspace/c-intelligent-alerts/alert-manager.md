# 경고 관리

경고 관리자에서 기존 경고를 관리할 수 있습니다. 태그 지정, 이름 변경, 삭제 등과 같은 경고에 대한 다양한 관리 작업을 수행할 수 있습니다.

경고 관리자에서 기존 경고를 관리하는 방법에 대한 자세한 내용은 [경고 관리자](/help/components/c-alerts/alert-manager.md)를 참조하십시오.
