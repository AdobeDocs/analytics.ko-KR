# 프로젝트 목차

목차는 각 프로젝트에서 사용할 수 있습니다. 이를 통해 프로젝트 내에서 패널과 시각화 사이를 빠르게 이동할 수 있습니다. 이 기능은 많은 패널과 시각화가 포함된 더 큰 프로젝트를 볼 때 특히 유용합니다.

프로젝트에 대한 목차를 보려면 다음과 같이 하십시오.

1. Analysis Workspace에서 목차를 보려는 프로젝트로 이동합니다.

1. 왼쪽 탐색에서 목차 아이콘을 선택합니다 ![목차 아이콘](assets/toc-icon.png).

   프로젝트에 대한 목차가 표시되고 확장됩니다.

1. 프로젝트 내의 패널 또는 시각화로 이동하려면 목차 내의 섹션을 클릭합니다.
