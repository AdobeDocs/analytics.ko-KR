# 시각화 개요

작업 영역에는 막대 차트, 도넛 차트, 히스토그램, 라인 차트, 지도, 스캐터플로 등과 같이 데이터를 시각적으로 표현할 수 있는 여러 가지 시각화가 있습니다. 각 시각화에는 관리할 수 있는 자체 설정이 있습니다. 자세한 내용을 보려면 시각화의 이름을 클릭합니다.

YouTube 비디오:분석 [작업 공간의 시각화](https://www.youtube.com/watch?v=b1zLEywRa6w&amp;index=39&amp;list=PL2tCx83mn7GuNnQdYGOtlyCu0V5mEZ8sS) 유형(2:57)

| 시각화 이름 | 설명 |
|---|---|
| [영역](/help/analyze/analysis-workspace/visualizations/area.md) | 선 그래프와 유사하지만 선 아래에 색상 영역이 있는 경우 여러 지표가 있고 둘 이상의 지표가 교차하여 표현된 영역을 시각화하려는 경우 영역 그래프를 사용합니다. |
| [막대](/help/analyze/analysis-workspace/visualizations/bar.md) | 하나 이상의 지표에서 다양한 값을 나타내는 세로 막대를 표시합니다. |
| [글머리 기호 그래프](/help/analyze/analysis-workspace/visualizations/bullet-graph.md) | 다른 성능 범위(목표)와 비교하거나 측정하려는 값을 표시합니다. |
| [집단 테이블](/help/analyze/analysis-workspace/visualizations/cohort-table/cohort-analysis.md) | *`cohort`*&#x200B;은 지정된 기간 동안 공통적인 특성을 공유하는 사람들의 그룹입니다. 집단 분석은 예를 들어 집단이 브랜드에 어떻게 참여하는지를 알려고 할 때 유용합니다. 트렌드 변경 사항을 쉽게 찾아 응답할 수 있습니다. |
| [도넛](/help/analyze/analysis-workspace/visualizations/donut.md) | 파이 차트와 유사하게, 이 시각화는 데이터를 전체 부분 또는 세그먼트로 보여줍니다. |
| [폴아웃](/help/analyze/analysis-workspace/visualizations/fallout/fallout-flow.md) | 폴아웃 보고서는 방문자가 페이지의 사전 정의된 순서를 떠나고(폴아웃) 계속 따라가는(폴스루) 위치를 보여줍니다. |
| [흐름](/help/analyze/analysis-workspace/visualizations/c-flow/flow.md) | 웹 사이트와 앱의 고객 경로를 보여줍니다. |
| [자유 형식 테이블](/help/analyze/analysis-workspace/visualizations/freeform-table.md) | 자유 형식 테이블은 데이터 테이블일 뿐만 아니라 대화형 시각화이기도 합니다. |
| [히스토그램](/help/analyze/analysis-workspace/visualizations/histogram.md) | 막대 그래프는 막대 차트와 비슷하지만, 숫자를 범위(버킷)로 그룹화합니다. |
| [가로 막대](/help/analyze/analysis-workspace/visualizations/horizontal-bar.md) | 하나 이상의 지표에서 다양한 값을 나타내는 가로 막대를 표시합니다. |
| [라인](/help/analyze/analysis-workspace/visualizations/line.md) | 일정 기간 동안 값이 변경되는 방식을 보여주기 위해 행을 사용하여 지표를 나타냅니다. 라인 차트는 시간이 차원으로 사용될 때만 사용할 수 있습니다. |
| [맵](/help/analyze/analysis-workspace/visualizations/map-visualization.md) | 모든 지표(계산된 지표 포함)의 시각적 맵을 작성할 수 있도록 해줍니다. |
| [산포도](/help/analyze/analysis-workspace/visualizations/scatterplot.md) | 차원 값과 최대 3개 지표 간의 관계를 표시합니다. |
| [요약 번호](/help/analyze/analysis-workspace/visualizations/summary-number-change.md) | 선택한 셀에 따라 이 시각화에서는 총계와 요약이 표시됩니다. |
| [요약 변경](/help/analyze/analysis-workspace/visualizations/summary-number-change.md) | 선택한 셀에 따라 이 시각화는 셀을 서로 비교합니다. |
| [텍스트](/help/analyze/analysis-workspace/visualizations/text.md) | 사용자 정의 텍스트를 작업 영역에 추가할 수 있습니다. |
| [트리맵](/help/analyze/analysis-workspace/visualizations/treemap.md) | 계층적(트리 구조) 데이터를 중첩된 사각형 세트로 표시합니다. |
| [벤](/help/analyze/analysis-workspace/visualizations/venn.md) | 구성 요소에서 최대 3개의 세그먼트와 하나의 지표를 드래그하여 벤 다이어그램을 만들 수 있습니다. |

## 시각화 패널 {#section_DC07F032FBEF4046A40F7B95C28DA018}

To display the Visualizations panel, click **[!UICONTROL Visualizations]** in the side panel.

![단계 결과](assets/visualizations.png)

Adobe Analytics를 사용하는 경우 대부분의 시각화 유형(예: 면적, 막대, 도넛 및 라인 차트)에 익숙할 것입니다. 그러나 분석 작업 공간에서는 시각화 설정과 대화형 기능을 사용하여 새로운 또는 고유한 여러 시각화 유형을 제공합니다.

## 시각화 설정 {#section_D3BB5042A92245D8BF6BCF072C66624B}

액세스하려면 [!UICONTROL Visualization Settings]시각화를 드래그하여 [!UICONTROL Freeform Panel]표시한 다음 [!UICONTROL Visualization Settings] 톱니바퀴 아이콘을 클릭합니다.

>[!IMPORTANT]
>
>표시되는 시각화 설정은 시각화에 따라 다릅니다. 일부 설정이 모든 시각화에 적용되는 것은 아닙니다. 또한 일부 고급 설정은 막대 그래프 설정과 같은 특정 시각화에 **대해서만** 표시됩니다 [](/help/analyze/analysis-workspace/visualizations/histogram.md#section_09D774C584864D4CA6B5672DC2927477).

![](assets/visualization_settings.png)

| 설정 | 설명 |
|--- |--- |
| 백분율 | 값을 백분율로 표시합니다. |
| 100% 스택 | 스택형 영역 또는 스택형 막대 또는 스택형 가로 막대 시각화의 이 설정은 차트를 &quot;100% 누적&quot; 시각화로 바꿉니다. 예: ![](assets/stacked_100_percent.png) |
| 범례 표시 | 요약 번호/요약 변경 사항 시각화에 대한 필터 세부 사항 텍스트를 숨길 수 있습니다. |
| 최대 항목 제한 | 시각화가 표시하는 항목 수를 제한할 수 있습니다. |
| Y축 0에 연결 | 차트에 표시된 모든 값이 0보다 상당히 높은 경우 차트 기본값은 y축 NON-ZERO의 하단으로 설정됩니다. 이 확인란을 선택하면 y축이 0으로 설정되며 차트가 다시 그려집니다. |
| 표준화 | 지표를 등분 비례에 강제 적용합니다. |
| 이중 축 표시 | 지표가 두 개일 경우에만 적용됩니다. 왼쪽(한 지표에 대해)과 오른쪽(다른 지표에 대해)에 y축을 놓을 수 있습니다.  |
| 예외 항목 표시 | 선 그래프와 자유 형식 테이블을 강화하여 데이터 예외 항목을 표시합니다. |

## 시각화 만들기 아이콘 {#section_9C11D9DEDC42413AA53E69A71A509DFC}

If you are not sure which visualization to pick, click the **[!UICONTROL Create Visual]** icon in any table row. 이 아이콘은 테이블 행 위로 마우스를 가져가면 나타납니다. 이를 클릭하면 분석 작업 공간이 데이터에 가장 적합한 시각화를 추측하도록 표시됩니다. 예를 들어 최대 3개의 세그먼트를 선택한 경우 벤 다이어그램을 만듭니다. 3개 이상의 세그먼트에 대해 막대 차트를 만듭니다. 다른 유형의 데이터의 경우 선 그래프 등을 만들 수 있습니다.

![](assets/create-visual.png)

## 시각화/패널 메뉴를 마우스 오른쪽 단추로 클릭 {#section_05B7914D4C9E443F97E2BFFDEC70240C}

시각화나 패널 헤더 옆에 있는 설정을 마우스 오른쪽 단추로 클릭하면 그래프와 컨텍스트에 맞는 설정에 액세스할 수 있습니다. 다음 설정 중 일부 또는 전체를 사용할 수 있습니다.

![](assets/right-click_menu.png)

| 설정 | 설명 |
|--- |--- |
| 복사된 시각화/패널 삽입 | 복사한 요소를 프로젝트 내의 다른 위치에 붙여넣거나 완전히 다른 프로젝트에 붙여넣을 수 있습니다(&quot;삽입&quot;). |
| 시각화/패널 복사 | 시각화 또는 패널을 마우스 오른쪽 단추로 클릭하여 복사할 수 있습니다. |
| 시각화/패널 복제 | 현재 시각화를 정확히 복제하여 수정할 수 있습니다. |
| 모든 패널 축소 | 모든 프로젝트 패널을 축소합니다. |
| 패널의 모든 시각화 축소 | 이 프로젝트 패널에서 모든 시각화를 축소합니다. |
| 모든 패널 확장 | 모든 프로젝트 패널을 확장합니다. |
| 패널에서 모든 시각화 확장 | 이 프로젝트 패널에서 모든 시각화를 확장합니다. |
| 설명 편집 | 시각화/패널에 대한 텍스트 설명을 추가(또는 편집)합니다. 이 설명은 프로젝트 >  프로젝트 정보 및 설정에 표시됩니다 . |
| 패널 링크 가져오기 | 프로젝트 내의 특정 패널로 다른 사람을 안내할 수 있습니다. |
| 시각화 링크 가져오기 | 이 링크를 복사하고 공유하여 다른 사람을 이 시각화로 직접 보낼 수 있습니다. 사용자는 로그인해야 합니다. |
| 다시 시작 | (흐름, 벤, 히스토그램에 적용) 현재 시각화에 대한 구성을 삭제하고 다시 구성할 수 있는 새 패널을 엽니다. |

## 범례 레이블 편집 {#section_94F1988CB4B9434BA1D9C6034062C3DE}

시각화 범례(폴아웃, 영역, 스택된 영역, 막대, 스택형 막대, 도넛, 막대 그래프, 가로 막대, 스택형 가로 막대, 선, 산포 및 벤)에서 시리즈 이름을 변경하여 시각 효과를 보다 소비적으로 만들 수 있습니다.

범례 편집은 다음 항목에 적용되지 **않습니다** .트리 맵, 글머리 기호, 요약 변경 또는 번호, 텍스트, 자유 형식, 막대 그래프, 집단 또는 흐름 시각화.

라인 차트에서 범례 레이블을 편집하려면

1. 범례 레이블 중 하나를 마우스 오른쪽 단추로 클릭합니다.
1. 클릭 **[!UICONTROL Edit Label]**.

   ![](assets/edit-label.png)

1. 새 레이블 텍스트를 입력합니다.
1. Press **[!UICONTROL Enter]** to save.

다음은 이 항목의 [비디오에 대한 링크](https://www.youtube.com/watch?v=mry3vDrTml0&amp;index=61&amp;list=PL2tCx83mn7GuNnQdYGOtlyCu0V5mEZ8sS)입니다.
