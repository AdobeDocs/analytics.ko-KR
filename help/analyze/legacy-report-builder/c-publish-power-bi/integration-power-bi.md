# Power BI용 Adobe Analytics 콘텐츠 팩

{{legacy-arb}}

Adobe Analytics 콘텐츠 팩에서는 사이트 트래픽 및 사용자 차원에 대한 인사이트를 제공하는 사전 빌드된 Power BI 대시보드와 Power BI 보고서 세트를 제공합니다. 재공된 대시보드와 보고서를 사용하거나, 사용자 정의하여 가장 중요한 정보를 강조 표시할 수 있습니다. 데이터는 하루에 한 번 자동으로 새로 고쳐집니다.

3개의 콘텐츠 팩을 사용할 수 있습니다.

* Adobe Analytics
* Adobe Analytics 모바일 앱 분석
* Adobe Analytics 트래픽 분석

자세한 내용은 [Power BI용 Adobe Analytics 콘텐츠 팩](https://powerbi.microsoft.com/ko-kr/documentation/powerbi-content-pack-adobe-analytics/)을 참조하십시오.
