# 보고서 유형 개요

{{legacy-arb}}

사이트 지표, 사이트 컨텐츠 및 비디오와 같은, 데이터 요청에 대한 기본 보고서 유형을 선택할 수 있습니다.

스프레드시트 셀 범위에 대해 하나의 기본 보고서 유형만 선택할 수 있습니다. 이전에 만든 요청을 편집하는 경우 요청에서 다른 설정을 다시 구성하지 않고 [!UICONTROL 요청 마법사: 1단계] 창에서 보고서 유형을 변경할 수 있습니다.

자동 완성 검색 창을 사용하여 보고서를 검색할 수 있습니다. 이 컨트롤에서 보고서를 선택하면 일치하는 노드가 트리 보기에서 자동으로 선택됩니다.
