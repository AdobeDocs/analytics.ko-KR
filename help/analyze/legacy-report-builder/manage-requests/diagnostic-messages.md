# 진단 메시지

{{legacy-arb}}

진단 메시지에서는 새로 고칠 때 요청의 상태에 대한 정보를 알 수 있습니다.

![요청을 새로 고쳤음을 나타내는 녹색 확인 표시의 아이콘](assets/icon_notice_success.gif) 이 통합 문서를 연 후 요청이 정상적으로 새로 고쳐졌음을 나타냅니다. 진단 열에는 요청에 사용된 시간, 요청을 매핑하는 시간, 요청이 이전에 캐시한 정보를 사용하는지 여부 등 요청에 대한 다른 정보가 표시됩니다.

![요청을 새로 고치지 못했음을 나타내는 느낌표가 있는 빨간색 삼각형의 아이콘.](assets/icon_notice_warn.gif) 이 경고는 요청이 성공적으로 새로 고쳐지지 않을 때 표시됩니다. 진단 열에 오류의 가능한 원인을 나타내는 오류가 표시됩니다. 선택한 오류 메시지에 대한 도움말을 보려면 [문제 해결 섹션](/help/analyze/legacy-report-builder/troubleshoot.md)을 확인하십시오.
