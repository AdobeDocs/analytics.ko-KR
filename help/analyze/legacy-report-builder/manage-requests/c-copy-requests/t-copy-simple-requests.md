# 단순 요청 복사

{{legacy-arb}}

참조 요청이 아닌 간단한 요청을 복사합니다. 단순 요청은 다른 요청 또는 셀의 내용에 대한 참조가 없는 요청입니다.

[참조 요청](/help/analyze/legacy-report-builder/manage-requests/c-copy-requests/t-copy-referential-requests.md)에서는 데이터 필터나 관계 필터와 같은 매개 변수에 대한 입력으로 셀의 값들을 사용합니다. 이러한 필터는 일치 또는 트렌드를 사용하며, 이전 요청의 결과나 입력 셀이라고 하는 셀의 사용자 입력 내용을 기반으로 합니다.

단순 요청을 복사하려면

1. 올바른 요청을 만듭니다.
1. 요청이 매핑된 셀들 중 하나를 마우스 오른쪽 단추로 클릭하거나 요청이 들어 있는 셀 영역을 선택합니다.

   요청이 다루는 셀 그룹에서 복사할 셀을 선택할 때 일관성을 유지하십시오. 선호하는 선택은 요청이 다루는 셀 세트의 맨 위 셀과 맨 왼쪽 셀이며 왼쪽에서 오른쪽으로 작업합니다. Excel 스프레드시트의 열 수백 개와 행 수 천 개를 오른쪽 및 아래쪽 확장에 사용할 수 있기 때문입니다. 요청과 연결된 셀 세트의 맨 오른쪽 또는 맨 아래 셀에서 요청 복사를 시작하기로 결정하는 경우, 붙여넣을 셀이 스프레드시트의 왼쪽 또는 맨 위 테두리를 넘어 확장되는 경우 시스템에서 요청을 붙여넣을 수 없습니다.
1. **[!UICONTROL 요청 복사]**&#x200B;를 선택합니다.
1. 스프레드시트의 다른 부분에서 빈 셀(요청이 들어 있지 않은 셀)을 마우스 오른쪽 단추로 클릭합니다.

   이미 만든 요청이 손실되거나 손상되지 않도록 요청이 들어 있는 셀을 현재 요청으로 매핑된 셀에 붙여 넣을 수 없습니다. 요청이 들어 있는 셀을 복사하거나 잘라내면 바로 가기 메뉴에서 요청이 들어 있는 셀(또는 셀 집합)을 마우스 오른쪽 단추로 클릭할 때 [!UICONTROL 요청 붙여넣기] 옵션을 사용할 수 없습니다. 요청이 겹치지 않도록 다른 셀을 붙여넣기 작업의 대상으로 선택해야 합니다. 이는 붙여넣을 요청이 있는 단일 셀을 선택했는지 아니면 요청이 포함된 셀 영역을 선택했는지 여부에 적용됩니다.
1. **[!UICONTROL 요청 붙여넣기]**&#x200B;를 클릭합니다.

   원본 요청의 사본은 원본 요청과 상대적인 위치에 있는 셀에 배치됩니다.

   >[!NOTE]
   >
   >요청만 복사되며 셀의 내용은 복사되지 않습니다. 요청을 기반으로 하지 않지만 표 열 머리글이나 행 식별자 등 셀에 표시되는 데이터를 이해하는 데 관련된 다른 정보가 있는 경우 Excel의 표준 명령 복사 및 붙여넣기를 사용합니다.

   Excel에서는 셀 내용을 복사하고 요청을 복사하는 데 다른 클립보드를 사용하기 때문에 요청이 아닌 셀 내용을 모두 복사한 다음 요청/붙여넣기 복사/요청을 직렬로 수행하여 요청을 복사할 수 있습니다. 그러나 스프레드시트에서 요청에 서식을 적용한 다음 복사하여 붙여넣으면 Report Builder은 원래 서식(예: 테두리, 글꼴 등)을 재생합니다 을 클릭합니다.

   요청을 붙여넣기 전에 클립보드에 복사하거나 잘라낸 요청을 수정하면 클립보드에서 요청이 제거됩니다. 따라서 요청을 원래 상태로 유지하려면 요청을 복사한 시간과 붙여넣은 시간 사이에 요청을 수정하지 마십시오.
