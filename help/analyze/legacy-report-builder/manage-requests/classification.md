# 분류

{{legacy-arb}}

분류 분류는 분석 보고 데이터를 관련 속성에 매핑하는 데 사용됩니다. 분류는 다양한 용도로 사용할 수 있지만 가장 일반적으로 캠페인 추적 코드(내부 및 외부 모두) 및 제품 ID를 분류하는 데 사용됩니다.

예를 들어 [!UICONTROL Campaign] 보고서에 [!UICONTROL 관리자] 분류와 [!UICONTROL 콘센트] 분류가 있을 수 있습니다. 그런 다음 특정 계층 구조에 따라 이러한 분류의 순서를 지정할 수 있으며 각 보고서 값 그룹을 하위 그룹으로 분류할 수 있습니다.

보고서 세트의 다른 보고서에 액세스하는 것과 동일한 방식으로 분류에 액세스하면 차원이 그룹 내에 최상위 수준 보고서로 표시됩니다. 보고서에 분류가 있는 경우 동일한 위치에서 보고서 및 모든 해당 분류를 수집하기 위한 메뉴 하위 그룹이 만들어집니다. 보고서는 분류 중 목록의 마지막 항목으로 표시됩니다.

[분류](/help/components/classifications/classifications-overview.md)를 참조하십시오.
