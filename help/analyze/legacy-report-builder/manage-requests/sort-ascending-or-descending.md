# 오름차순 또는 내림차순 정렬

{{legacy-arb}}

요청 관리자에 나열된 요청을 오름차순이나 내림차순으로 정렬할 수 있습니다.

표 머리글에서 열 이름을 클릭합니다. 열은 테이블 헤더에 삼각형 아이콘을 표시하여 정렬 순서를 표시합니다. 삼각형이 아래쪽을 가리키면 정렬 순서는 내림차순(영숫자 데이터, 9-1 및 Z-A)이고, 위쪽을 가리키면 정렬 순서는 오름차순(영숫자 데이터, 1-9 및 A-Z)입니다.

기본 정렬 순서는 오름차순입니다.
