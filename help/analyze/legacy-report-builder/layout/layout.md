# 레이아웃 - 요청 마법사 2단계

{{legacy-arb}}

[!UICONTROL 요청 마법사: 2단계] 창을 사용하면 출력에 대한 보고서 레이아웃, 머리글 서식 지정 및 차원 필터링을 구성할 수 있습니다. 피벗 레이아웃을 사용하여 요청을 구성할 수 있으며 이것은 Excel 피벗 테이블을 만드는 방식과 유사합니다.

또는 [사용자 지정 레이아웃](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md)을 사용하여 보고서 데이터를 표시할 셀을 수동으로 선택할 수 있습니다.
