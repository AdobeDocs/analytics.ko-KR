# 트래픽 및 상거래 지표를 동일한 요청에 통합

별도의 요청을 만드는 대신 트래픽 및 상거래 지표 유형을 하나의 요청에 결합할 수 있습니다.

트래픽 및 상거래 지표는 요청 마법사: 2단계에서 동일한 지표 범주로 표시됩니다. 그룹화된 범주의 레이블은 **[!UICONTROL 표준]**&#x200B;입니다.
