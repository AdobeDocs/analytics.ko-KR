# 개별 시작 또는 종료 날짜

시작 및 종료 날짜의 약자

* cd-1d = 어제
* cm-1d = 지난 달의 마지막 날
* cw-1w = 지난 주의 첫째 날
* cw+1w-1d = 현재 주의 마지막 날
* cd-2d = 2일 전
* cw-2w = 2주 전 주의 첫째 날

날짜를 미래 날짜로 설정하면 미래 날짜에 대한 값이 반환되지 않습니다.
