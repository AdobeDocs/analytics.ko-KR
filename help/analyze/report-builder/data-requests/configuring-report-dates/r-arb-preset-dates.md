# 요청 마법사 정의 - 사전 설정 날짜

사전 설정 날짜는 공통으로 사용된 날짜 또는 저장한 날짜를 선택할 수 있도록 해줍니다. 요청 마법사: 1단계에서  사전 설정 날짜를 선택합니다. 다음은 Report Builder의 사전 설정 날짜에 대한 필드 정의입니다.
