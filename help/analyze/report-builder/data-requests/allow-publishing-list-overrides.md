# 게시 목록 무시 허용

보고서를 예약할 때 배포에 사용할 게시 목록을 선택할 수 있습니다.

게시 목록은 Analytics 관리 도구에서 설정됩니다.

분석 [참조에서 게시](https://marketing.adobe.com/resources/help/ko_KR/reference/publishing_list.html) 목록 관리자를 참조하십시오.

이 기능을 활성화하려면 [!UICONTROL Request Wizard: Step 1] 창으로 이동합니다.

활성화하면 게시 [!UICONTROL Allow Publishing List Override]목록에서 각 수신자에게 지정된 보고서 세트가 이 요청에 대한 보고서 세트를 대체합니다. 또한 통합 문서에 여러 보고서 세트가 포함되어 있는 경우 게시 목록에 연결된 보고서 세트 ID가 사용됩니다.

셀에서 선택하는 보고서 세트에는 이 옵션을 사용할 수 없습니다.

>[!NOTE] 예약된 보고서를 여러 게시 목록에 전송하면 보고서는 각 목록에 대해 한 번씩 실행됩니다. 변수 보고서 세트는 게시 목록에 지정된 보고서 세트로 대체됩니다.
