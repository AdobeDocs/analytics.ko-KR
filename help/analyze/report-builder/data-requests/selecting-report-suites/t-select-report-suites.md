# 보고서 세트를 선택합니다

Excel에서 보고서 세트를 선택하는 데 도움이 되는 절차.

1. In Excel, click **[!UICONTROL Add-Ins]**.
1. In the toolbar, click **[!UICONTROL Create]**.
1. [!UICONTROL 요청 마법사에서: 1 단계] 창 아래에서 보고서 세트 아래의 [!UICONTROL 보고서 세트를 선택합니다].
