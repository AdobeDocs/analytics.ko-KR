# 보고서 유형 선택

데이터 요청에 대한 기본 보고서 유형을 선택하는 절차.

1. 창으로 [!UICONTROL Request Wizard: Step 1] 이동합니다.
1. Select a report suite from the **[!UICONTROL Report Suite]** menu.
1. 데이터 요청에 대한 기본 보고서 유형을 선택합니다.
