# 오버레이 세부정보

오버레이 상세 정보는 링크 오버레이의 맨 위를 가리키면 표시됩니다.

오버레이 상세 정보에는 해당 링크에 대해 추적한 다음의 값들이 표시됩니다.

* 지표
* 원시 값
* 등급
* 백분율 값
* 링크 ID
* 영역
* 페이지에 있는 링크 수 보고서에 표시
