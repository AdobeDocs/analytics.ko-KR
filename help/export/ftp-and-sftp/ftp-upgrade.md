# Adobe FTP 및 SFTP 서버 업그레이드

Adobe FTP 및 SFTP 서버로 업그레이드하면 특정 서버 설정을 사용하도록 작성된 자동화 스크립트(정기적으로 데이터를 다운로드하거나 업로드하는 데 사용됨)에 영향을 주는 새로운 구성이 도입될 수 있습니다.

예를 들어, 성공 상태를 사용하여 특정 작업을 트리거하는 스크립트에 영향을 주는 새로운 성공 상태가 도입됩니다. Adobe은 이러한 구성 변경 사항을 사용자에게 사전 통지합니다. Adobe에서 FTP 서버 업그레이드가 임박했음을 알리는 경우 자동화 스크립트가 제대로 작동하는지 확인하십시오.
