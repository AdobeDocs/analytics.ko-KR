# FTP 제한 및 데이터 유지

모든 Adobe FTP 계정은 데이터 저장 공간이 10GB로 제한됩니다. 파일은 이러한 FTP 계정에서 90일간 유지됩니다.

## 동시 연결 제한

* 소스 IP당 서버에 대한 동시 연결 수가 50개 이내입니다.
* 소스 IP당 서버에 60초 이내에 20개 이하의 새로운 연결
