# Data Warehouse 게재된 보고서

>[!NOTE]
>
>다음 정보는 FTP 및 SFTP 대상 유형과 관련되어 있습니다. FTP 및 SFTP는 기존 대상 유형입니다. Data Warehouse 보고서를 구성할 때는 보다 안전한 클라우드 대상 유형을 사용해야 합니다. Data Warehouse 보고서에 대한 클라우드 대상 유형을 구성하는 방법에 대한 자세한 내용은 [Data Warehouse 요청 만들기](/help/export/data-warehouse/create-request/t-dw-create-request.md)를 참조하십시오.

Data Warehouse를 사용하면 특정 질문에 따라 원시 데이터의 고급 데이터 관계를 표시하는 보고서를 요청할 수 있습니다.

이 경우 FTP 배달은 [!UICONTROL 고급 배달 옵션] 설정의 일부로 사용할 수 있습니다. FTP를 사용하려면 기존 내부 FTP 계정이나 Adobe FTP 계정이 필요합니다. Adobe은 data warehouse 인터페이스를 통해 FTP 계정을 자동으로 설정하지 않습니다. FTP 배달은 파일 크기와 처리 중인 보고서 수에 따라 즉시 이루어져야 합니다.

FTP 제한 및 데이터 유지에 대한 자세한 내용은 [FTP 제한 및 데이터 유지](/help/export/ftp-and-sftp/ftp-limits.md)를 참조하십시오.
