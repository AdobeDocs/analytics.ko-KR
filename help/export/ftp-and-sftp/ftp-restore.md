# 삭제된 FTP 또는 SFTP 계정 및 데이터 복원

FTP 또는 SFTP 계정에서 삭제된 데이터는 Adobe에서 복원할 수 없습니다.

사용하려는 FTP 또는 SFTP 계정이 비활성화된 경우 추가 옵션에 대해 논의할 수 있는 Adobe 계정 팀에 문의하십시오.
