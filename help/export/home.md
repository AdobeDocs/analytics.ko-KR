# Analytics 내보내기 안내서

![배너](../../assets/doc_banner_export.png)

본 가이드에서는 Adobe Analytics에서 데이터를 가져오는 방법을 간략하게 설명합니다. 이러한 서비스에는 다음이 포함됩니다.

* **데이터 피드:** 원시 데이터의 시간별 또는 일별 내보내기를 받습니다. 모든 행은 개별 히트이며 모든 열은 변수입니다. 데이터 피드는 일반적으로 FTP 사이트로 전송됩니다.
* **데이터 웨어하우스:** 요청 마법사를 사용하여 스프레드시트의 데이터 출력을 검색합니다. 데이터 웨어하우스에서는 다른 처리 아키텍처를 사용하여 임의 개수의 행과 임의 개수의 고유한 값을 허용합니다.
