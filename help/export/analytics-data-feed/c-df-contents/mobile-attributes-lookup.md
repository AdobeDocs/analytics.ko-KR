# 모바일 속성 조회

모바일 속성은 히트에서 모바일 장치의 속성을 표시합니다. 이 파일은 다음과 같은 경우에 나타납니다.

* 동적 조회를 사용할 수 있습니다. 조직의 지원 위임자는 원하는 피드 ID를 사용하여 고객 지원 센터에 연락하여 동적 조회를 활성화할 수 있습니다.
* 이 `mobile_id` 열은 포함됩니다.
* 열은 `user_agent` 제외됩니다. DeviceAtlas와의 라이선스 계약 때문에 이 제외가 필요합니다.
