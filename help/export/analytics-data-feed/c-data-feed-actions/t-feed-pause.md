# 피드 일시 정지

활성 피드를 일시 정지할 수 있습니다.

피드를 일시 정지하면 현재 작업이 처리를 완료하고 새 작업이 시작되지 않습니다. [피드 상태]가 "비활성"으로 변경됩니다.

1. 하나 이상의 피드를 선택합니다.
1. **[!UICONTROL 일시 중지를]**&#x200B;클릭합니다.

   For information about activating a paused feed, see [Activate a Feed](t-feed-activate.md).
