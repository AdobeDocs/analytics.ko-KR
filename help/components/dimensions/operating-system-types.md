# 운영 체제 유형

운영 체제 유형 차원은 특정 버전에 관계없이 방문자가 사용한 지배적인 OS를 보여줍니다. 이 차원은 가장 일반적인 특정 운영 체제와 버전뿐만 아니라 방문자가 사용하는 일반적인 OS 플랫폼을 파악하는 데 유용합니다.

## 이 차원을 데이터로 채우기

이 차원은 Adobe 내부의 조회 테이블을 참조합니다. 조회 값은 이미지 요청에 있는 `User-Agent` HTTP 헤더를 기반으로 합니다. AppMeasurement 라이브러리를 사용하는 경우 (Adobe Experience Platform Launch 등을 통해) 이 차원은 즉시 작동합니다. 

## 차원 항목

차원 항목에는 사용되는 운영 체제의 유형이 포함됩니다. 예로는 `"Microsoft Windows"`, `"Apple Macintosh"`, `"Google Android"` 및 `"Apple iOS"`가 있습니다.
