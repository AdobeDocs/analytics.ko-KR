# 마케팅 채널 세부 사항

&#39;마케팅 채널 세부 정보&#39; [차원](overview.md)은(는) [&#39;마지막 터치 채널 세부 정보&#39;](last-touch-detail.md) 차원과 동일하게 작동합니다. Adobe에서는 사용자 지정 속성을 사용하는 보고서에 사용하기 위해 이 차원을 도입했습니다. 마지막 터치가 사용자 지정 속성에 정확하지 않아 사용자 지정 속성 마케팅 채널 보고서를 본 사용자가 &#39;마지막 터치&#39; 레이블과 혼동했었습니다. Adobe는 분석가가 오해의 소지가 있는 “마지막 터치” 차원 레이블 없이 사용자 지정 속성 마케팅 채널 보고서를 작성할 수 있도록 일반 “마케팅 채널 세부 정보” 차원을 도입했습니다.
