# 모바일 보기 수

> [!NOTE] 이 보고서는 2017년 9월 21일의 유지관리 버전과 함께 폐기되었으며 더 이상 데이터를 반환하거나 보고에 사용할 수 없습니다.

모바일 보기 개수 지표 대신 &quot;모바일 장치에서 방문 수&quot; 세그먼트를 적용하는 것이 좋습니다.
