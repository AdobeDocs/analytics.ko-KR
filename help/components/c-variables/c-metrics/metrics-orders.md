# 주문

선택된 기간 동안의 웹 사이트 주문 수. 다른 지표별로 개별 기간을 분류하여 해당 기간 동안 가장 많은 주문 수를 기록한 항목(제품, 캠페인 등)을 볼 수 있습니다.

| 사용 | 설명 |
|---|---|
| 트래픽 | 해당 없음 |
| 전환 | *`Order`* 은 구매 이벤트가 설정되는 횟수입니다. |
| Ad Hoc Analysis | *`Order`* 은 구매 이벤트가 설정되는 횟수입니다. |
