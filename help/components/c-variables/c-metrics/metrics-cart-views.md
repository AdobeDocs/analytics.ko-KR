# 장바구니 보기

고객이 장바구니의 내용물을 보는 이벤트 이 값은 scView 이벤트에서 가져옵니다.

| 사용 | 설명 |
|---|---|
| 트래픽 | 해당 없음 |
| 전환 | 설정할 [!UICONTROL 장바구니 보기] 이벤트 인스턴스. 장바구니의 품목을 볼 때 발생합니다. |
| Ad Hoc Analysis | 설정할 [!UICONTROL 장바구니 보기] 이벤트 인스턴스. 장바구니의 품목을 볼 때 발생합니다. |
