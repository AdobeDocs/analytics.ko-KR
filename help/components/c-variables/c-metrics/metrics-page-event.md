# 이벤트 페이지

페이지 이벤트는 비표준 이미지 요청의 이미지 요청 데이터로 구성됩니다. 비표준 이미지 요청의 소스는 다운로드 링크, 종료 링크 및 사용자 지정 링크 추적입니다.
