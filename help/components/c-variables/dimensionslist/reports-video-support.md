# 비디오 지원

모바일 장치를 지원하는 비디오 형식 유형에 따라 그룹화합니다. 예를 들어, 방문자의 모바일 장치가 .mp4를 지원하는 경우 비디오 지원 보고서는 사이트에 액세스할 때마다 최소 하나씩 증가합니다. 전화기가 여러 이미지 형식을 지원하는 경우(.mp4, .wmv 등) 보고서에서 각 그룹이 증가됩니다. 이로 인해 그룹 합계는 보고서 하단에 표시된 총계보다 더 클 수 있습니다.
