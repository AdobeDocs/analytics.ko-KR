# 장치 번호 전송

장치 번호 전송이 지원되는지 여부를 기준으로 모바일 장치를 그룹화합니다. 장치 번호 전송 지원을 확인할 수 없는 장치를 위해 알 수 없음 그룹도 포함합니다.
