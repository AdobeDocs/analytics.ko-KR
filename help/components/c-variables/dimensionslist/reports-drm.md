# DRM(디지털 권한 관리)

모바일 장치를 지원하는 DRM(디지털 권한 관리) 유형에 따라 그룹화합니다. DRM 그룹은 전방 잠금, 통합 배달, 개별 배달 및 알 수 없음을 포함합니다.
