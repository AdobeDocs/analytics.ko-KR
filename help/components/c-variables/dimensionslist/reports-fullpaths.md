# 전체 경로

방문자가 가장 많이 사용하는 전체 웹 사이트 방문 경로를 표시합니다.

전체 경로 보고서는 8페이지의 방문 깊이로 제한됩니다. 이 깊이보다 긴 방문자 움직임은 기록되지 않습니다.
