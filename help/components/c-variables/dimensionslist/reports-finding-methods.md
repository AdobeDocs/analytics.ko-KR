# 검색 방법

전환 성공 이벤트에 대해 받은 크레딧을 표시합니다. 예를 들어 검색 엔진이 구매하는 방문자를 해당 사이트로 안내하는 경우, 검색 방법은 검색 엔진이 참조에 대한 크레딧을 받는 방법을 지정합니다.

구성에 대해서는 [검색 방법](/help/admin/admin/finding-methods.md)을 참조하십시오.
