# 색상 깊이

모바일 장치 히트 수를 지원되는 색상의 수로 그룹화합니다. 이 보고서/측정기준은 모바일 장치를 사용한 총 사이트 방문자 수를 보여주며 해당 모바일 장치에 구성된 색상의 수에 따라 방문자를 그룹화합니다. 예를 들어 방문자의 휴대 전화기가 24색을 지원하는 경우 보고서는 24색에 해당하는 라인 항목을 증가시킵니다.
