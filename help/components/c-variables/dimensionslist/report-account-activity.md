# 계정 활동

보고서 세트 트래픽에 대한 요약 데이터를 제공합니다.

계정 활동 보고서는 Adobe Analytics 2018 여름 릴리스의 서버 호출 사용 기능으로 대체됩니다. 계정 활동 보고서는 2018년 8월 9일에 영구적으로 제거됩니다. 2018년 8월 9일 이후 보고서 세트 트래픽에 대한 요약 데이터를 보려면 서버 호출 사용 기능을 사용합니다.
