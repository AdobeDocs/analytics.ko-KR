# 서버

특정 서버에서 호스팅하는 페이지를 그룹화할 수 있습니다. 웹 사이트의 모든 서버를 나열하고 그 중에서 액세스 빈도가 가장 높은 서버를 알려줍니다. 예를 들어 두 개의 서버로 웹 사이트를 호스팅하는 경우 어느 한 서버의 노출 횟수가 더 많은지 확인하면 유용할 수 있습니다.
