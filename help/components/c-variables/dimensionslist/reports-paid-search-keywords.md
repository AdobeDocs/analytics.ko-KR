# 유료 검색 키워드

사이트를 찾는 데 사용한 각 유료 검색 키워드의 분류를 표시합니다. 목록 위에 있는 열 제목을 클릭하여 페이지 보기 수 또는 검색 키워드별로 이 목록을 정렬할 수 있습니다. 사이트에 대한 검색 결과를 보려면 검색 키워드 옆에 있는 확대경을 클릭합니다.
