# 이벤트까지 남은 시간

방문 중에 선택된 전환 이벤트가 발생하기까지 경과된 시간을 표시합니다. 방문을 시작한 이미지 요청과 특정 전환 이벤트를 발생시킨 이미지 요청 간 시간 차이를 확인할 수 있습니다. 이 보고서는 사람들이 사이트에 도착할 때부터 성공 이벤트(예: 양식 제출 또는 주문하기)를 발생할 때까지 소요되는 시간을 파악하는 데 유용합니다.
