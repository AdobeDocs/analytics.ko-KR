# 방문 횟수

각 방문자가 사이트를 방문하는 횟수를 추적하여 방문자 충성도를 평가하는 데 도움이 됩니다. 선택한 기간 동안 사이트를 처음 방문하는 방문자 또는 20번째 방문하는 방문자의 방문이 더 많은지 여부를 확인할 수 있습니다.

방문 번호는 선택한 날짜 범위가 아닌 방문자의 라이프타임을 기반으로 합니다.
