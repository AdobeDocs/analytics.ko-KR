# 브라우저 및 FTP 가져오기

분류 템플릿 파일에 적절한 데이터를 복사한 후 Adobe 데이터 수집 서버에 가져올(다운로드) 수 있습니다.

>[!NOTE] 새로운 데이터를 업로드하기 전에 내보내기 기능을 사용하여 기존 분류 데이터를 다운로드할 필요가 없습니다. 새 데이터 행을 포함하는 새 파일만 업로드하면 됩니다. 이 방법은 성능을 향상시킵니다.

See [General File Structure](/help/components/c-classifications2/c-classifications-importer/c-saint-data-files.md) for important information about the structure and limitations of uploaded files.

문제 해결 도움말은 [일반적인 분류 업로드 문제](https://helpx.adobe.com/kr/analytics/kb/common-saint-upload-issues.html)를 참조하십시오.
