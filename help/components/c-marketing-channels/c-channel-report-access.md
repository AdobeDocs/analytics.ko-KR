# 보고서 액세스 권한

마케팅 채널 보고서를 실행하고, 보고서에 대한 제한적인 관리자 권한 및 사용자 그룹 권한을 부여하는 방법에 대해 설명합니다.

마케팅 채널 보고서를 보려면 사용자 그룹에 적절한 권한을 부여하십시오. Adobe Experience Cloud의 권한 시스템은 그룹을 기반으로 합니다. 즉, 사용자 그룹이 보고서에 대한 액세스 권한을 결정합니다.
