# 미확인된 사용자

‘미확인된 사용자’ 지표는 CDA로 식별할 수 없는 방문자의 수를 계산하는 [디바이스 간 분석](../cda/overview.md) 지표입니다.

## 이 지표의 계산 방법

새 방문자가 결합 서비스에 여전히 알려지지 않은 경우(특정 사용자에게 할당될 수 없는 디바이스에서 제공되는 히트) 이 지표는 증가합니다.
