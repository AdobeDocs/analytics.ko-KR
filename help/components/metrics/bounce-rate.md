# 바운스 비율

&#39;바운스 비율&#39; 지표는 해당 페이지의 방문 수와 비교하여 정확히 하나의 히트가 포함된 방문 비율을 보여줍니다. 이 지표를 사용하여 바운스 비율이 가장 높은 차원 값을 이해하거나 시간에 따라 사이트의 집계된 총 바운스 비율을 볼 수 있습니다.
