# 확인된 사용자

&#39;확인된 사용자&#39; [지표](overview.md)은(는) CDA로 확인된 방문자의 수를 계산하는 [교차 장치 분석](../cda/overview.md) 지표입니다.

## 이 지표의 계산 방법

새 방문자가 결합 서비스로 식별되는 경우(히트는 특정 새 사용자와 연계됨) 이 지표는 증가합니다.
