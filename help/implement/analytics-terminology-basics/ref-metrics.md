# 지표

지표는 보기 수, 클릭스루 횟수, 다시 로드 횟수, 평균 체류 시간, 판매량, 주문 수, 매출액 등과 같은 방문자 활동에 대한 수량 정보입니다.

지표는 보고서의 기초가 되며, 데이터 관계를 보고 이해하는 데 도움이 됩니다. 웹 사이트에 대한 여러 데이터 세트를 나란히 비교하는 데 사용됩니다.
