# s.linkName

사용자 지정, 다운로드 또는 종료 링크 보고서에 표시되는 이름입니다. 100자에서 잘리며, 어떤 문자열로도 설정할 수 있습니다. 이미지 요청의 `pev2` 매개 변수입니다.

If set with `linkType`, an image request will be sent as a download, custom or exit link.
