# 데이터 레이어 만들기

데이터 레이어는 일반적으로 구현 내의 모든 변수 값을 포함하는 사이트의 JavaScript 개체입니다. 사이트 레이아웃이 변경될 때 문제를 방지하려면 데이터 레이어를 사용하는 것이 좋습니다.
