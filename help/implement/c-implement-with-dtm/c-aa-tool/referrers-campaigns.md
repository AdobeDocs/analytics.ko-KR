# 레퍼러 및 캠페인

DTM 인터페이스에서 쿼리 문자열을 사용할지 아니면 값(데이터 요소에서 끌어올 수 있음)을 사용할지 선택합니다.

![](assets/dtm-queryparam.png)

인터페이스에서 직접 쿼리 문자열을 입력하거나 캠페인을 추적할 다른 방법이 있는 경우 별도의 데이터 요소를 참조할 수 있습니다.
