# Javascript 라이브러리 파일

JavaScript 라이브러리 파일은 초기 로드 후에 사용자의 브라우저에서 캐시되도록 되어 있어서 다운로드해야 하는 데이터 양을 제한합니다.

페이지별 변수는 HTML 조각에 삽입해야 하며, 모든 다른 변수는 JavaScript 라이브러리 파일에 삽입해야 합니다.
