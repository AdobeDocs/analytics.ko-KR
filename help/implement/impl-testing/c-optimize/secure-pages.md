# 보안 페이지

보안 페이지(https://로 로드된 페이지)는 이미지 요청을 암호화하고 총 다운로드 시간에 추가합니다.

보안 페이지는 50-75% 정도의 오버헤드를 이미지 요청에 추가할 수 있습니다.
