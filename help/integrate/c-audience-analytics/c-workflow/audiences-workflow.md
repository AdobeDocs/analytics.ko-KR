# Audience Analytics 워크플로 개요

대상자 데이터를 제공, 공유 및 사용하는 방법입니다.

| 단계 # | 작업 설명 | 참고 |
|--- |--- |--- |
| 1단계 | 서버측 전달을 켭니다. | 관리 사용 안내서에서 [서버측 전달](/help/admin/tools/manage-rs/edit-settings/general/c-server-side-forwarding/ssf.md)을 참조하십시오. |
| 2단계 | Adobe Audience Manager에서 Analytics 대상을 구성합니다. | [Analytics 대상 만들기](https://experienceleague.adobe.com/docs/audience-manager/user-guide/features/destinations/experience-cloud-destinations/create-analytics-destination.html?lang=ko-KR)를 참조하십시오. |
| 3단계 | [Analytics에서 데이터를 사용](/help/integrate/c-audience-analytics/c-workflow/use-audience-data-analytics.md)합니다. | Analysis Workspace 및 관련 기능(세그먼트 비교, 흐름, 벤 등)에서 새로운 Adobe Audience Manager 대상 차원을 사용할 수 있습니다. <br>대상 차원은 Report Builder, Data Warehouse 및 데이터 피드에서도 사용할 수 있습니다. <br>**참고:** 데이터 피드에는 대상자 ID만 포함되고, 대상자 이름은 포함되지 않습니다. |
