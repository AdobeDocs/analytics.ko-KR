# Adobe Analytics에 대한 기술 설명서 업데이트

2019년 1월 이후에는 Adobe Analytics 설명서 세트에 대한 컨텐츠 업데이트입니다.

* 자세한 내용 [!UICONTROL Customer Journey Analytics]은 [여기](https://docs.adobe.com/content/help/ko-KR/analytics-platform/using/cja-landing.html)로 가세요
* For a comprehensive list of new Adobe Analytics features, Admin notices, and bug fixes, see the latest [Experience Cloud Release Notes](https://docs.adobe.com/content/help/ko-KR/release-notes/experience-cloud/current.html).
* 새로운 기능에 대한 자세한 내용은 도구 사용자 안내서의 [분석 작업 공간의](/help/analyze/analysis-workspace/new-features-in-analysis-workspace.md) 새로운 기능을 참조하십시오.
* 오디오 및 비디오용 Adobe Analytics에 대한 자세한 내용은 [Analytics 에서 오디오 및 비디오 측정](https://docs.adobe.com/content/help/ko-KR/media-analytics/using/media-overview.html)을 참조하십시오.
