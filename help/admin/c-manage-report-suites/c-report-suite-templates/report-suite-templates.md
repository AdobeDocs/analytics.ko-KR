# 보고서 세트 템플릿 개요

보고서 세트 템플릿은 여러 유형의 보고서 세트에 대한 가장 일반적인 설정을 구성합니다.

새 보고서 세트를 만들 때 시간을 절약하기 위해 설계한 보고서 세트 구성과 유사한 템플릿을 선택할 수 있습니다. 보고서 세트 템플릿을 선택하더라도 구현이 제한되지는 않습니다. 보고서 세트를 만든 후 보고서 세트 템플릿으로 작성된 구성을 변경할 수 있습니다.
