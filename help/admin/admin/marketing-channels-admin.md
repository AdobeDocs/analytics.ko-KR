# 마케팅 채널

마케팅 채널은 일반적으로 사이트에 (채널을 통한) 방문자가 도착하는 방식에 대한 통찰력을 제공하는 데 사용됩니다. 추적할 채널과 추적 방법에 따라 마케팅 채널 처리 규칙을 생성하고 사용자 지정할 수 있습니다.
