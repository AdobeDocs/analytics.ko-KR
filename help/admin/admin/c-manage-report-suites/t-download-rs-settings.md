# 보고서 세트 설정 다운로드

선택한 보고서 세트의 모든 설정이 포함된 Excel 스프레드시트를 생성하려면 다음을 수행합니다.

1. **[!UICONTROL 관리]****[!UICONTROL >]**&#x200B;보고서 세트를 클릭합니다.
1. [!UICONTROL 보고서 세트] 테이블에서 보고서 세트를 선택합니다.
1. **[!UICONTROL 다운로드]**&#x200B;를 클릭합니다.

   스프레드시트 파일을 직접 열거나 저장했다가 볼 수 있습니다.
