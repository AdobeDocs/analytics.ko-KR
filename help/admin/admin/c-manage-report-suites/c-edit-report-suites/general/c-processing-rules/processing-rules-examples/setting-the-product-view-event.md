# 제품 개요 페이지에서 제품 보기 이벤트 설정

이벤트를 페이지 값 또는 변수의 특정 값을 기준으로 설정할 수 있습니다.

| 규칙 세트 | 값 |
|---|---|
| 조건 | 페이지 이름이 제품 개요와 동일한 경우 |
| 작업 | 제품 보기 이벤트를 사용자 지정 값 1로 설정 |
