# 데이터 소스

데이터 소스 기능을 사용하면 오프라인 소스의 데이터를 Analytics로 가져올 수 있습니다. 가져온 데이터는 기본적으로 수집된 데이터와 유사하게 처리 및 처리될 수 있습니다.

데이터 소스를 사용하여 다음을 수행합니다.

* 오프라인 데이터를 웹 사이트에서 수집한 데이터와 통합합니다.
* 웹 서버, 콜 센터 또는 기타 시스템에서 데이터를 수집합니다. 그런 다음 CX Enterprise 로 가져와서 조직의 모든 경로에서 전환을 확인합니다.
* SearchCenter을 통한 오프라인 리드 이행을 기반으로 키워드 입찰 관리를 최적화합니다.

Data Sources가 보고서 세트에서 활성화되면 보고에 통합할 파일을 업로드할 수 있도록 FTP 위치가 제공됩니다. 업로드되면 Adobe 서버가 데이터를 자동으로 처리하여 보고서 세트에 통합합니다.

[데이터 원본 도움말](/help/import/data-sources/overview.md)을 참조하세요.
