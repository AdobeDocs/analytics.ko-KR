# 기본 지표

[!UICONTROL **설정 편집**] > [!UICONTROL **개별 보고서 세트 설정**] > [!UICONTROL **기본 지표**]

사용자가 사용자 지정 지표 세트를 선택하지 않는 경우 Analytics는 모든 보고서 및 프로젝트에 기본(즉시 사용 가능한) 지표 세트를 표시합니다. 선택된 지표가 연관된 보고서 세트의 모든 사용자에게 표시됩니다. 한 번에 하나의 보고서 세트의 기본 지표만 업데이트할 수 있습니다. 이러한 설정은 기본 지표가 표시되는 순서를 보장하지 않습니다.

자세한 내용은 다음 문서를 참조하십시오.

* [Analysis Workspace의 지표](/help/analyze/analysis-workspace/components/apply-create-metrics.md)

* [계산된 지표](/help/components/calculated-metrics/cm-overview.md)
