# Adobe Analytics에서 사용하는 IP 및 도메인

일부 방화벽 구성은 Adobe 데이터 수집 서버나 데이터에 액세스하는 서버에서 오는 IP 주소를 차단합니다. 이 범위 목록을 사용하여 조직의 방화벽 설정을 변경하여 액세스 허용 및 조직 내에서 데이터를 전송할 수 있습니다.

>[!IMPORTANT]
>
>Adobe은 이 문서를 최신 상태로 유지하는 데 최선을 다하지만 IP 범위 목록이 동일하게 유지되는지 보장할 수 없습니다. 성장 및 사업 확대, Adobe의 IP 주소 공간 변경, 인터넷 서비스 제공업체 중단 등의 변경 사항이 있을 수 있습니다.
