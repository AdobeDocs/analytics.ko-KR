# 데이터 커넥터 이메일 통합 준비{#preparing-for-the-data-connectors-email-integration}

데이터 커넥터 통합 마법사를 실행하여 데이터 커넥터 통합을 구현하기 전에 수행해야 하는 작업 및 정보입니다.

Adobe Analytics®는 조직의 온라인 전략 및 마케팅 이니셔티브에 대한 실행 가능한 실시간 인텔리전스를 제공합니다. Analytics는 emarsys와 함께 행동 타깃팅을 기반으로 리마케팅을 제공합니다. emarsys 및 Analytics 웹 분석 데이터를 매끄럽게 통합하여 타겟 마케팅을 향상시키고 캠페인 효과를 파악할 수 있습니다.

이메일 계정에 대한 통합을 구성하려면 먼저 다음을 수행해야 합니다.

* 데이터 커넥터 마법사에 입력되기 때문에 emarsys AccountCode에 대해 알아보십시오.
* 통합 작업과 관련된 알림을 받는 데 사용할 이메일 주소를 알아봅니다.
* 수신자 ID를 캡처하는 데 사용할 사용자 지정 변수를 식별하거나 만듭니다.
* 메시지 ID를 캡처하는 데 사용할 사용자 지정 변수를 식별하거나 만듭니다(종종 캠페인 변수).
* 바운스, 클릭, 열기, 보내기 및 가입 해지를 수집하는 데 사용할 사용자 지정 이벤트를 식별하거나 만듭니다.
